# Required Libraries

This addon requires the following Ace3 libraries to be embedded in this directory:

## Required Libraries

- **LibStub** - Library stub for embedding
- **CallbackHandler-1.0** - Callback handling for events
- **AceAddon-3.0** - Main addon framework
- **AceConsole-3.0** - Chat commands
- **AceDB-3.0** - Database/saved variables management
- **AceDBOptions-3.0** - Profile management UI
- **AceEvent-3.0** - Event handling
- **AceConfig-3.0** - Configuration framework
- **AceConfigDialog-3.0** - Configuration dialog
- **AceConfigRegistry-3.0** - Configuration registry

## Optional Libraries

- **LibDataBroker-1.1** - Data broker for display addons
- **LibDBIcon-1.0** - Minimap button support
- **LibQTip-1.0** - Advanced tooltip support (optional, for enhanced tooltips)

## Installation

Download these libraries from:
- Ace3: https://www.wowace.com/projects/ace3
- LibDataBroker-1.1: https://www.wowace.com/projects/libdatabroker-1-1
- LibDBIcon-1.0: https://www.wowace.com/projects/libdbicon-1-0

Or use a package manager like packager to automatically embed them during release.
